# Generate-Posters
利用puppeteer根据html模板生成海报。

可直接部署腾讯云serverless，运行环境：Nodejs12.16。



项目目录：

├── README.md
├── index.js
├── package.json
├── render.js						   // 主渲染js
└── template  						// 模板目录	
    └── message.html
